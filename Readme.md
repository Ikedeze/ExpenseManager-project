# Personal Expense Tracker
A simple Java console application for tracking daily expenses.
## Features
- Add, remove, and view expenses
- Filter by category, date, or keyword
- Save and load data from a file
- View total and average expenses
- Category-wise spending summary

## How to Run

1. Compile all Java files:
javac *.java

2. Run the main program:   
java Main

## Example Menu
1. Add Expense
2. Remove Expense
3. View All Expenses
4. Save / Load Expenses
5. Filter / Search
6. View Summary
0. Exit

## Technologies Used
- Java 12
- File I/O
- LocalDate API
- Collections (ArrayList, HashMap)
